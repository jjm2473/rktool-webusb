rkdeveloptool Web 版

说明

仅支持 Chrome/Edge 等基于 Chromium 的浏览器

不建议在 Windows 下使用，因为 Windows 下有 Rockchip 官方的图形刷机工具。且可能需要安装 WinUSB 驱动，驱动又可能跟 Rockchip 驱动冲突。自行评估风险

一般刷机步骤：

  1. 设备进入 MASKROM 或 Loader 模式（通常是开机时按住某个按键，具体请搜索设备型号+MASKROM/Loader）
  2. 设备通过 USB 连接到电脑（通常是设备的 Type-C 接口，也可能是 A 口）
  3. 选择 Loader 文件（ MASKROM 模式需要）
  4. 选择固件
  5. 点击刷写固件按钮，此过程可能会弹出设备授权请求（MASKROM 模式最多两次），如果设备列表为空，请检查设备是否正确连接（如 USB 口是否正确，设备是否进入正确模式）
  6. 刷写完成后在“电源”菜单中选择相应操作（如重启或关机），或手动操作设备
如需检查设备状态，点击“选择 Rockchip 设备”按钮，列表为空说明没有设备连接或设备未进入正确模式。
如需刷写到不同的存储器，例如有 SPIROM 和 eMMC 的设备希望写入 eMMC，先加载Loader，然后切换存储，最后再刷写固件。
